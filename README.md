# Simple cli timer

```sh
$ timer -h
Simple cli timer

Usage: timer [OPTIONS]

Options:
  -H, --hours <HOURS>      Hours [default: 0]
  -m, --minutes <MINUTES>  Minutes [default: 0]
  -s, --seconds <SECONDS>  Seconds [default: 0]
  -l, --line-mode          Print elapsed time on single line
  -h, --help               Print help
  -V, --version            Print version
```
